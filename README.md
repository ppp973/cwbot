# 🚀 CareerWill Premium Telegram Bot

## Features
- ✅ Extract any CareerWill batch
- ✅ View all batches with clickable IDs
- ✅ Multiple batch extraction
- ✅ Live progress tracking
- ✅ DRM detection
- ✅ Premium UI with emojis

## Commands
- `/start` - Welcome message
- `/help` - Help guide
- `/about` - Bot info
- `/cwextractfree` - Extract a batch
- `/allbatches` - View all batches

## Deploy
1. Fork repository
2. Connect to Render
3. Deploy as Background Worker
